# BaronyRu
Русификатор для игры barony

ИГРА БУДЕТ ВЫЛЕТАТЬ, ЕСЛИ НЕ СДЕЛАТЬ:
В файлах игры в свойствах файла compiled_books.json по пути "C:\Program Files (x86)\Steam\steamapps\common\Barony\books\compiled_books.json" надо поставить галочку - только для чтения! ИНАЧЕ ИГРА БУДЕТ КРАШИТСЯ!!!

Мод переводит почти всю игру на русский. В некоторых местах текст выезжает за рамки(особенно заметно в обучении).
Таблички, которые выводят текст сверху, перевести не получится, так как они вшиты в игру.
Пишите если надо где-то что-то подправить, также если текст переведён коряво.
