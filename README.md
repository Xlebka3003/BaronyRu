# BaronyRu v5.0.2
Русификатор для игры Barony v5.0.2

Установка:

1. Скачиваем репозиторий.
2. Создаем в корне игры папку "mods" 
3. В папке "mods" создаем папку например "BaronyRu"
4. Разархивируем все файлы в папку "BaronyRu" 

ВНИМАНИЕ!!!

Если вам нужны книги с историями: в файлах игры в свойствах файла compiled_books.json "Barony\books\compiled_books.json" надо поставить галочку - только для чтения! ИНАЧЕ ИГРА БУДЕТ КРАШИТСЯ!!!

Если вам книги не нужны, просто удалите папку "books" этого мода "Barony\mods\BaronyRu\books".

Мод переводит почти всю игру на русский. В некоторых местах текст выезжает за рамки(особенно заметно в обучении).
Таблички, которые выводят текст сверху, перевести не получится, так как они вшиты в игру.
Пишите если надо где-то что-то подправить, также если текст переведён коряво.
